This file contains all the scripts used for configuring run grids, data analysis, 
and plotting that went into the production version of https://arxiv.org/abs/2308.06666,
"The Impact of Angular Momentum Loss on the Outcomes of Binary Mass Transfer"
by Reinhold Willcox, Morgan MacLeod, Ilya Mandel, & Ryosuke Hirai.

If you run into issues, please contact reinhold.willcox (at) gmail.com.
